# greening-india-backend


## 📌 Description
Backend API for tracking tree plantation and carbon offset.

## 🚀 Features
- Add tree plantation data
- View all plantations
- Get total trees and carbon stats

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB

## 📂 API Endpoints
- POST /api/trees → Add plantation
- GET /api/trees → Get all data
- GET /api/stats → Get analytics

## ⚙️ Setup
git clone <repo-link>
cd greening-india-backend
npm install
node server.js

## 🌍 Database
Supports:
- MongoDB Local
- MongoDB Atlas
